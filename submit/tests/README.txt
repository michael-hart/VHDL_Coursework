The tests in this directory are described as follows: 
sample_host_commands1.txt: default test given with project.
sample_host_commands2.txt: tests every command possible and writes to all pixels.
sample_host_commands3.txt: tests 20 repeats of each command, clears screen in all directions.
sample_host_commands4.txt: tests every possible pair of commands.
sample_host_commands.txt: concatenation of all previous tests.
